# TUI Rust Portfolio Template

This project is a one-page portfolio template with a simple terminal-style UI, built for a backend developer profile focused on Rust, security, and performance.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- framer-motion

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Customize Content

- Update hero, about, projects, and contact content in app/page.tsx.
- Replace the skills array and projects array with your own data.
- Update metadata title and description in app/layout.tsx.

## Production

```bash
npm run build
npm run start
```

## Motion Notes

- Uses lightweight reveal animation for sections and cards.
- Automatically respects reduced-motion preferences.
